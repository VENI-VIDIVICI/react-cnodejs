### react-cnodejs

基于react全家桶复刻CNode社区的WebApp，使用**create-react-app**脚手架搭建，糅合semantic和antd-mobile做UI搭建，后续会继续改进

特色：

* 拥抱新特性，使用**React 16.8 Hooks** API全面编写组件，并封装多个基础Hook
* 自行实现常见交互，包括无限滚动加载等
* 提升用户体验，列表切换引入缓存，文章详情页预加载，秒刷文章
